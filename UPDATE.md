# 更新记录

## 2026-07-14：自动二次纠偏

本次更新在原有的纸张检测与透视矫正之后，新增了一层精细的自动歪斜纠正。

### 新增功能

- 自动分析发票中的文字基线和表格横线，检测透视矫正后残留的小角度倾斜。
- 在可靠范围内自动旋转拉正，最大检测范围为左右各 7°。
- 旋转产生的空白区域使用白色填充，避免透明或黑色边角。
- JPEG 与 PDF 均使用纠偏后的结果，原始图片不会被覆盖。
- 批量处理时，每张图片都会单独判断角度；线条不足或可信度不够时保持原图方向，避免误矫正。
- 命令行新增 `--no-deskew` 参数，可在特殊情况下关闭二次纠偏。

### 当前处理流程

1. 识别纸张四角。
2. 裁切并完成透视矫正。
3. 检测文字行和表格线的残余倾角并精细纠偏。
4. 进行保留原彩色的扫描增强。
5. 输出高质量 JPEG 与 A4 PDF。

### 使用方式

安装或更新后，Finder 中的操作方式不变：

```text
选中图片 → 右键 → 快速操作 → 发票自动裁剪（图片+PDF）
```

已有用户在项目目录重新运行以下命令即可更新本机版本：

```bash
./install.sh
```
